# 100 Prompt Marcelus - Salute Cittadini

## Descrizione

Questa cartella raccoglie prompt per situazioni sanitarie comuni dei cittadini in Italia/Europa: lunga attesa in Pronto Soccorso, sovraffollamento, indicazioni vaghe, medico di famiglia, continuita assistenziale, referti, liste d'attesa, prevenzione e comunicazione sanitaria.

## Utilita

Aiuta a preparare domande, riassumere sintomi, capire quando chiedere rivalutazione, raccogliere documenti e comunicare meglio con professionisti sanitari, senza sostituire il giudizio clinico.

## Guardrail sanitario

I prompt non fanno diagnosi, non prescrivono terapie e non sostituiscono medico, pediatra, farmacista, infermiere, 112/118 o Pronto Soccorso. In caso di sintomi gravi o peggioramento improvviso va chiamata l'emergenza.
