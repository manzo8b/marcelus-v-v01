# 50 Prompt Marcelus - Prevenzione Cittadini

## Descrizione

Questa cartella raccoglie prompt per prevenzione pratica: blackout, caldo estremo, bollino rosso, anziani, bambini piccoli, alcol, farmaci, sicurezza domestica, condominio, quartiere e comunicazioni ai cittadini.

## Utilita

Aiuta cittadini, famiglie, caregiver, amministratori e piccole comunita a costruire piani semplici, verificabili e coerenti con fonti ufficiali.

## Guardrail

I prompt non sostituiscono 112/118, Protezione Civile, Comune, ASL, medico o gestori dei servizi. Ogni dato locale, numero, ordinanza o soglia tecnica deve essere verificato su fonte ufficiale.
