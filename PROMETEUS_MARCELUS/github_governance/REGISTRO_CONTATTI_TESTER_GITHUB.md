# Registro contatti tester GitHub

## Obiettivo

Raccogliere 10 contatti tester per Prometeus Marcelus in modo ordinato, prudente e non invasivo.

Questo registro serve a evitare messaggi casuali e a trasformare ogni contatto utile in feedback, Issue o miglioramento del progetto.

## Link da condividere

Link principale:

`https://github.com/manzo8b/marcelus-v-v01/releases/tag/v0.5.0-starter-kit`

Messaggio guida:

```text
Ciao, sto raccogliendo feedback su Prometeus Marcelus, uno Starter Kit italiano open source per trasformare richieste confuse in prompt piu chiari, prudenti e verificabili.

Se ti interessa AI pratica, prompt engineering o educazione digitale, mi sarebbe utile un feedback concreto:

https://github.com/manzo8b/marcelus-v-v01/releases/tag/v0.5.0-starter-kit
```

## Criteri per scegliere un contatto

Un contatto e interessante se almeno una di queste condizioni e vera:

- pubblica o cura prompt;
- lavora su AI, LLM o automazioni;
- scrive guide educative;
- partecipa a repository su AI safety;
- si occupa di data analysis;
- lavora in compliance, risk management o privacy;
- usa GitHub per documentazione o formazione;
- ha interesse per strumenti in italiano;
- lavora con piccoli business, food and beverage o formazione;
- potrebbe portare un caso reale.

## Cosa non fare

- Non inviare lo stesso messaggio a decine di persone.
- Non aprire Issue promozionali nei repository altrui.
- Non chiedere stelle come primo obiettivo.
- Non promettere risultati.
- Non coinvolgere persone su casi sanitari, legali o finanziari senza prudenza.
- Non raccogliere dati personali non necessari.
