# GitHub Governance - Projects, Discussions, Pull Requests

## Obiettivo

Trasformare `marcelus-v-v01` da archivio di prompt a progetto organizzato, verificabile e migliorabile.

## Projects

Usare un Project privato come cabina di regia operativa.

### Colonne consigliate

- Inbox
- Da valutare
- In lavorazione
- In revisione
- Pubblicato
- Da collegare a LinkedIn
- Archivio

### Card iniziali

- Training dataset Prometeus Marcelus
- Standard macroaree SUPERBIBLIOTECA
- Pagina Start Here per utenti non tecnici
- LinkedIn: raccogliere 10 tester
- Workflow Pull Request per modifiche importanti

### Routine settimanale

Ogni venerdi:

1. chiudere card completate;
2. spostare card in revisione;
3. scegliere massimo 3 priorita della settimana;
4. trasformare un avanzamento in post LinkedIn;
5. aggiornare la roadmap.

## Discussions

Attivare Discussions quando ci sono almeno 5-10 tester o persone interessate.

### Categorie consigliate

- Feedback Prometeus
- Idee nuove
- Prompt migliorati
- Problemi reali
- Roadmap
- Domande su GitHub
- Versioni pubbliche

### Quando usare Discussions

Usare Discussions per confronto aperto, domande, feedback e idee non ancora operative.

## Issues

Usare Issues per lavoro concreto e tracciabile.

Esempi:

- correggere un README;
- migliorare un prompt;
- fare privacy check;
- creare un caso test;
- preparare un post LinkedIn;
- aggiungere collegamenti a una macroarea.

## Pull Requests

Usare Pull Request anche lavorando da soli quando il cambio e importante.

### Quando aprire una PR

- nuova versione istruzioni Custom GPT;
- modifica della struttura SUPERBIBLIOTECA;
- training dataset;
- README principale;
- policy FREE / PREMIUM / PRIVATO;
- modifiche con rischio privacy.

### Template mentale PR

- Cosa cambia?
- Perche cambia?
- Quali file tocca?
- C'e rischio privacy?
- Come e stato verificato?
- Come si puo tornare indietro?

## Regola pratica

Issue = lavoro da fare.

Pull Request = lavoro fatto da controllare.

Discussion = idea o feedback da esplorare.

Project = quadro operativo.
