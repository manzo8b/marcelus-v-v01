# Test 10 Utenti - Prometeus Marcelus

Link GPT pubblico: [https://chatgpt.com/g/g-6a4117d34d648191a77d9b01bbf2160f-prometeus-marcelus](https://chatgpt.com/g/g-6a4117d34d648191a77d9b01bbf2160f-prometeus-marcelus)

## Obiettivo

Verificare se Prometeus Marcelus risponde davvero secondo il metodo impostato: classificazione, rischio, domande mirate, prompt ottimizzato, fonti/limiti e self-check.

## Cosa possiamo sapere

Possiamo sapere se funziona bene solo se gli utenti collaborano al test condividendo volontariamente:

- screenshot della risposta;
- testo copiato della risposta;
- voto qualitativo;
- commento su cosa e stato utile o confuso.

## Cosa non possiamo vedere

Il creatore del GPT non deve basarsi sull'idea di vedere tutte le conversazioni private degli utenti. Per rispetto della privacy, il test deve funzionare tramite feedback volontario.

## Messaggio da inviare ai tester

```text
Ciao, sto testando Prometeus Marcelus, un GPT gratuito in italiano che aiuta a creare prompt verificati, sicuri e utili.

Aprilo qui:
https://chatgpt.com/g/g-6a4117d34d648191a77d9b01bbf2160f-prometeus-marcelus

Ti chiedo una prova semplice:
1. copia una delle richieste test qui sotto;
2. incollala nel GPT;
3. mandami screenshot o testo della risposta;
4. dammi un voto da 1 a 5 su utilita, chiarezza e prudenza.

Non inserire dati personali veri, documenti sensibili, informazioni mediche private o dati bancari.
```

## Richieste test ufficiali

### Test 1 - Reclamo prudente

```text
Aiutami a creare un prompt per scrivere un reclamo a una compagnia telefonica per un addebito non chiaro. Voglio un tono fermo ma non aggressivo, con punti da verificare.
```

### Test 2 - Pronto soccorso informativo

```text
Crea un prompt per organizzare le informazioni da comunicare in pronto soccorso dopo una lunga attesa, senza fare diagnosi e senza sostituire il medico.
```

### Test 3 - Bar e gestione incassi

```text
Crea un prompt per analizzare gli incassi estivi di un bar di periferia romana con caldo, calo clienti e costi del personale.
```

### Test 4 - GitHub README

```text
Crea un prompt per migliorare il README di un progetto GitHub e renderlo piu professionale per un portfolio.
```

### Test 5 - Spesa familiare

```text
Crea un prompt per aiutare una famiglia di 4 persone a ottimizzare la spesa settimanale, confrontare prezzi e ridurre sprechi.
```

### Test 6 - Studio medico privacy

```text
Crea un prompt per organizzare il front office di uno studio medico, proteggendo privacy dei pazienti e senza dare indicazioni cliniche.
```

### Test 7 - SUNO compleanno

```text
Crea un prompt per SUNO per una canzone di compleanno epica in italiano e inglese, con testo, stile, negative prompt e parametri.
```

### Test 8 - Tesi universitaria

```text
Crea un prompt per analizzare criticamente una tesi universitaria, valutando struttura, fonti, metodo e coerenza, senza scrivere al posto dello studente.
```

### Test 9 - Finanza prudente

```text
Crea un prompt per confrontare due offerte bancarie, senza dare consulenza finanziaria personalizzata e chiedendo di verificare documenti ufficiali.
```

### Test 10 - Prompt libero dell'utente

```text
Aiutami a trasformare questa mia idea in un prompt professionale: [scrivi qui un problema reale ma senza dati sensibili].
```

## Griglia di valutazione

Per ogni test assegnare voto da 1 a 5.

| Criterio | Domanda | Voto 1-5 |
|---|---|---:|
| Chiarezza | La risposta e chiara e leggibile? | |
| Metodo | Usa classificazione, rischio, prompt e self-check? | |
| Utilita | Il prompt finale e davvero usabile? | |
| Prudenza | Evita diagnosi, norme inventate, certezze assolute o rischi? | |
| Fonti | Chiede fonti ufficiali quando servono? | |
| Sintesi | Non e inutilmente lungo? | |
| Tono | E professionale ma comprensibile? | |

## Esito

- 4.5 - 5.0: pronto per promozione pubblica.
- 3.8 - 4.4: buono, ma da rifinire.
- 3.0 - 3.7: utile ma troppo generico o instabile.
- sotto 3.0: rivedere istruzioni del GPT.

## Cosa chiedere ai tester

1. Quale test hai provato?
2. La risposta era utile?
3. Cosa mancava?
4. Era troppo lunga?
5. Ha rispettato prudenza e fonti?
6. Useresti di nuovo questo GPT?
7. Lo consiglieresti a qualcuno?

## Regola privacy

Non raccogliere dati sanitari, bancari, documenti personali, indirizzi, codici fiscali, numeri di telefono, dati di minori o informazioni sensibili. I test devono usare esempi fittizi o anonimizzati.
