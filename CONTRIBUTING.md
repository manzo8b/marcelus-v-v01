# Contribuire a Prometeus Marcelus

Grazie per l'interesse.

Il modo migliore per contribuire non e aggiungere molti prompt, ma portare problemi reali, test, correzioni e casi ben documentati.

## Come contribuire

Puoi contribuire con:

- un problema reale da trasformare in prompt;
- un test del Custom GPT;
- una correzione di un prompt poco chiaro;
- una segnalazione privacy o sicurezza;
- una fonte ufficiale utile;
- una proposta per Bibliotheca Promethea.

## Regole semplici

1. Non inserire dati personali reali.
2. Non pubblicare dati sanitari, legali, bancari, clienti o documenti riservati.
3. Non proporre prompt che promettono diagnosi, risultati finanziari o consulenze definitive.
4. Indica sempre se il contenuto e FREE, PREMIUM, PRIVATO o BOZZA.
5. Usa tono pratico, chiaro e verificabile.

## Formato consigliato per un caso

```text
Richiesta iniziale:
Contesto:
Obiettivo:
Cosa manca:
Rischio:
Prompt Prometeus migliorato:
Checklist:
Livello: FREE / PREMIUM / PRIVATO / BOZZA
```

## Prima di aprire una Pull Request

- Controlla che non ci siano dati sensibili.
- Aggiungi link interni se il file deve essere trovato dal README.
- Inserisci una checklist o un criterio di verifica.
- Mantieni il contenuto in italiano semplice.

## Cosa evitare

- Prompt generici senza contesto.
- Materiale premium o monetizzazione pubblicato per errore.
- Riferimenti a clienti, aziende o persone reali senza consenso.
- Claim legali, medici o finanziari non verificati.
- Liste enormi senza indice.
