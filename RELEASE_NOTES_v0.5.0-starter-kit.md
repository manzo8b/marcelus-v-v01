Questa release crea il primo package pubblico di Prometeus Marcelus.

Non e un GitHub Package tecnico.

E uno **Starter Kit editoriale e operativo** pensato per chi vuole capire, provare e testare Prometeus Marcelus senza leggere tutta la repository.

- punto di ingresso Start Here;
- guida "Usa Prometeus Marcelus";
- metodo operativo in 7 fasi;
- prompt master;
- checklist prudenza aree sensibili;
- sistema qualita Prometeus;
- playbook tester;
- casi reali prioritari Lotto 04;
- Promethea Economica v0.1;
- routine settimanale GitHub;
- strategia X e piano X free.

## Cosa non e

Questa release non e:

- una libreria software;
- un pacchetto npm;
- una consulenza legale, medica, fiscale o finanziaria;
- un prodotto commerciale finito;
- materiale premium.

## Limiti

I contenuti sono pubblici, educativi e operativi.

Non inserire dati personali nei test.

Non usare Prometeus o Promethea per prendere decisioni automatiche su salute, credito, lavoro, diritti, sicurezza o servizi essenziali.

Quando serve, verificare con fonti ufficiali o professionisti qualificati.
