# Roadmap Casi Reali Prometeus

## Fase 1 - Avvio

Obiettivo: raccogliere 10 problemi reali anonimi.

Azioni:

- pubblicare post LinkedIn;
- aprire GitHub Discussion;
- invitare contatti selezionati;
- evitare richiesta generica "testa il progetto";
- chiedere solo un problema concreto.

## Fase 2 - Prima trasformazione

Per ogni caso:

1. riformulare il problema;
2. creare prompt Prometeus;
3. aggiungere checklist;
4. indicare area SUPERBIBLIOTECA;
5. segnalare rischi privacy/AI Act;
6. salvare apprendimento.

## Fase 3 - Pubblicazione esempi

Pubblicare solo casi anonimi e autorizzati.

Formato:

- problema iniziale;
- prompt generico;
- prompt Prometeus;
- checklist;
- risultato atteso;
- cosa abbiamo imparato.

## Fase 4 - Miglioramento Custom GPT

Usare i casi per aggiornare:

- istruzioni Custom GPT;
- test reali;
- issue GitHub;
- prompt mancanti;
- macroaree SUPERBIBLIOTECA.

## Fase 5 - Monetizzazione prudente

Solo dopo casi reali:

- pacchetti settoriali;
- revisione prompt personalizzata;
- mini audit AI Act;
- template operativi;
- formazione base per piccoli team.

Non promettere risultati garantiti. Vendere metodo, chiarezza e verifica.
