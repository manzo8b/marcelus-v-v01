# Monetizzare Marcelus

## Sintesi

Marcelus puo' diventare un prodotto digitale a livelli: una parte gratuita su GitHub per autorevolezza e scoperta, una parte premium organizzata in manuali, template, consulenze, corsi e pacchetti verticali.

## Modello consigliato

1. **Free su GitHub**: prompt base, README, esempi, indice pubblico.
2. **Premium PDF/Notion/Gumroad**: prompt compilati, esempi reali, casi studio, checklist, schede operative.
3. **Pacchetti verticali**: F&B, bar periferia, GitHub per professionisti, spesa familiare, politica neutrale, SUNO, Terminal Mac.
4. **Servizi**: personalizzazione prompt per azienda, formazione 1:1, audit GitHub, creazione materiali didattici.
5. **Licenza professionale**: uso interno per enti, scuole, consulenti, ristoranti, bar, agenzie.

## Prodotti vendibili

- Marcelus Master Plus PDF
- Pacchetto F&B e ristorazione
- Pacchetto bar periferia romana
- Pacchetto GitHub pratico per professionisti
- Pacchetto spesa familiare e risparmio
- Pacchetto prompt SUNO
- Workshop online: costruisci il tuo prompt master
- Consulenza: trasformare un'attivita in libreria prompt

## Prezzi orientativi da testare

- Mini pack: 9-19 euro
- Pack verticale completo: 29-79 euro
- Manuale premium: 49-149 euro
- Workshop live: 49-199 euro a persona
- Consulenza personalizzata: 150-600 euro a progetto

## Strategia pratica

Mantieni GitHub come vetrina autorevole. Non vendere tutto subito: usa GitHub per dimostrare qualita, poi crea versioni premium piu comode, curate e applicate.

## Regola importante

Il valore non e' solo il prompt. Il valore e': contesto, esempi compilati, casi reali, checklist, spiegazione, aggiornamenti, supporto e personalizzazione.
