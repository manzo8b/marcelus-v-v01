# Guida Uso SUPERBIBLIOTECA_MARCELO

## Obiettivo

Questa guida serve a usare la struttura Marcelus come sistema professionale di classificazione dei prompt.

## Percorso Rapido

Se devi creare o archiviare un prompt:

1. Chiediti qual e il risultato pratico del prompt.
2. Scegli una delle 12 macroaree.
3. Scegli la sottocartella piu specifica.
4. Aggiungi nel prompt una riga di metadati con macroarea, sottocategoria, livello e rischio.
5. Se il prompt tocca salute, legge, finanza o sicurezza, aggiungi fonti e limiti.

## Livelli Consigliati

- FREE: prompt pubblici e divulgativi.
- PREMIUM: prompt piu lunghi, professionali o monetizzabili.
- PRIVATO: materiale strategico, monetizzazione, prezzi, clienti, procedure interne.
- BOZZA: materiale non ancora verificato.

## Metadati Minimi per Ogni Prompt

```text
Macroarea: [numero_nome]
Sottocategoria: [numero_nome]
Livello: FREE / PREMIUM / PRIVATO / BOZZA
Rischio: basso / medio / alto
Fonti richieste: si / no
Uso previsto: cittadino / professionista / impresa / formazione / creativo
```

## Regola per Prompt Ibridi

Se un prompt sembra appartenere a due cartelle:

- scegli la cartella dove verra usato davvero;
- cita la seconda area nei metadati;
- non duplicare il file in due posti.

## Esempi

Un prompt su caldo, anziani e 112 va in:
`08_SICUREZZA_CITTADINA_EMERGENZE/03_Prevenzione_Cittadini_Caldo_Blackout_e_Fragilita`

Un prompt su referto cardiologico va in:
`09_SALUTE_INFORMATIVA_CARDIOLOGIA/03_Farmaci_Referti_e_Domande_per_Cardiologo`

Un prompt su food cost di un bar va in:
`07_FOOD_BEVERAGE_RISTORAZIONE/05_Contabilita_Ristorativa_Food_Cost_Beverage_Cost`

Un prompt su README GitHub va in:
`04_TECH_PROGRAMMAZIONE_AI/04_GitHub_Pratico_Repository_e_README`
