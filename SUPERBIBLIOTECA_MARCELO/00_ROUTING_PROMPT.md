# Routing Prompt Marcelus

## Domanda Base

Dove devo mettere questo prompt?

Rispondi con questa sequenza.

## 1. Il prompt crea qualcosa di visuale, sito, brand o contenuto?

Usa `01_DESIGN_IDENTITA_WEB`.

## 2. Il prompt insegna, forma, valuta o produce materiale didattico?

Usa `02_FORMAZIONE_EDUCAZIONE_RICERCA`.

## 3. Il prompt serve a musica, SUNO, lyrics, album o cover?

Usa `03_MUSICA_GENERATIVA_SUNO`.

## 4. Il prompt riguarda codice, AI workflow, GitHub, Terminal o strumenti digitali?

Usa `04_TECH_PROGRAMMAZIONE_AI`.

## 5. Il prompt riguarda rischio, compliance, AML, privacy, HR o sicurezza aziendale?

Usa `05_RISK_COMPLIANCE_SICUREZZA_AZIENDALE`.

## 6. Il prompt riguarda soldi, banche, spesa, consumi, risparmio o dashboard?

Usa `06_FINANZA_BANCHE_CONSUMI`.

## 7. Il prompt riguarda bar, ristorazione, food cost, spiaggia o hospitality operativa?

Usa `07_FOOD_BEVERAGE_RISTORAZIONE`.

## 8. Il prompt riguarda cittadini, quartiere, emergenze, autostrada, prevenzione o prove?

Usa `08_SICUREZZA_CITTADINA_EMERGENZE`.

## 9. Il prompt riguarda salute informativa, cardiologia, PS, medico o referti?

Usa `09_SALUTE_INFORMATIVA_CARDIOLOGIA`.

## 10. Il prompt analizza politica, societa, partiti, bias o fonti pubbliche?

Usa `10_POLITICA_SOCIETA_ANALISI_NEUTRALE`.

## 11. Il prompt serve a vendere servizi, portfolio, clienti, preventivi, CV o monetizzazione?

Usa `11_BUSINESS_PERSONALE_MONETIZZAZIONE`.

## 12. Il prompt serve soprattutto a verificare fonti, norme, contratti, documenti o citazioni?

Usa `12_FONTI_VERIFICHE_DOCUMENTI_UFFICIALI`.

## Regola di Priorita

Quando un prompt ha piu anime, scegli in questo ordine:

1. Rischio alto: salute, legge, sicurezza, finanza, privacy.
2. Uso operativo concreto.
3. Dominio professionale.
4. Forma dell'output.

Esempio: un prompt per contestare una spesa bancaria non va in Business, ma in Finanza/Banche o Fonti/Verifiche se il cuore e normativo.
